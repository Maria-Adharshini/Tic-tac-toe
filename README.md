# Tic-tac-toe
Tic-Tac-Toe Game – Developed a responsive two-player Tic-Tac-Toe game using React, HTML, and CSS. Implemented game logic for player turns, customizable player symbols, winner and draw detection, and automatic game state updates using React components and state management.

Key skills demonstrated:

- React Components
- React Hooks (useState)
- State Management
- Event Handling
- Conditional Rendering
- HTML & CSS
- Game Logic
- Responsive UI Design
